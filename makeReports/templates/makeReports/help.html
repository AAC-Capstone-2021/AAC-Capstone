Question 1: When using the rich text editor, it does not allow me to submit because it is too long, even though it is not very long.
Answer: Copying and pasting from outside sources (especially Word) combined with certain browser add-ons that restrict Javascript can result in a large amount of hidden mark-up language. To fix, press the "Clean" button in the upper-left hand corner of the rich text editor.
